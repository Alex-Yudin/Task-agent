# Единая Google Таблица для «Орбиты» и ChatGPT

Google Таблица становится центральным реестром. Windows и Android сохраняют локальную SQLite-копию, поэтому продолжают работать без интернета, а после восстановления связи обмениваются изменениями с таблицей.

## 1. Создание таблицы

1. Загрузите `Orbita-Google-Sheets-Template.xlsx` в Google Drive.
2. Откройте файл через Google Таблицы и выберите **Файл → Сохранить как Google Таблицу**.
3. Откройте **Расширения → Apps Script**.
4. Удалите пример функции и вставьте содержимое `integrations/google-apps-script/Code.gs`.
5. В настройках проекта включите среду выполнения V8, если она ещё не включена.
6. Выберите функцию `setupOrbita`, нажмите **Выполнить** и разрешите доступ.
7. Сохраните показанный `Sync secret` в надёжном месте.

Если шаблон не использовался, `setupOrbita` самостоятельно создаст листы `Instructions`, `Projects` и `Tasks`.

## 2. Публикация защищённого API

1. В Apps Script нажмите **Deploy → New deployment**.
2. Выберите тип **Web app**.
3. Укажите **Execute as: Me**.
4. Укажите **Who has access: Anyone**.
5. Нажмите **Deploy** и скопируйте URL, оканчивающийся на `/exec`.

URL доступен из интернета, но чтение и запись данных выполняются только при наличии длинного случайного секрета. Не публикуйте секрет и не вставляйте его в общедоступные инструкции GPT.

## 3. Подключение Windows

1. Запустите `configure-google-sheets.cmd`.
2. Вставьте URL `/exec`.
3. Вставьте `Sync secret`.
4. Перезапустите `start-windows.cmd`.
5. В интерфейсе нажмите **Синхронизировать Google** для первой проверки.

После этого каждое изменение проекта или задачи автоматически ставится в очередь на выгрузку. Дополнительно «Орбита» проверяет изменения таблицы каждые пять минут.

## 4. Подключение Android

1. Откройте **Настройки → Синхронизация**.
2. Вставьте тот же URL `/exec`.
3. Вставьте тот же секрет.
4. Нажмите **Сохранить и синхронизировать**.

Android отправляет изменения после создания задачи, создания проекта и изменения статуса. Кнопка синхронизации запускает полный обмен вручную.

## 5. Подключение ChatGPT

1. Подключите Google Drive в ChatGPT и разрешите доступ к папке с таблицей.
2. Используйте инструкции из [CHATGPT_TASKS.md](CHATGPT_TASKS.md) в проекте или персональном GPT.
3. В первом сообщении укажите точное название таблицы и попросите прочитать листы `Projects` и `Tasks`.

ChatGPT не нужен секрет Apps Script: он работает с таблицей через разрешённое подключение Google Drive. Секрет нужен только приложениям «Орбита».

## Колонки

Основные поля задач: `id`, `projectId`, `projectTitle`, `parentTaskId`, `title`, `description`, `status`, `priority`, `dueAt`, `completedAt`, `author`, `createdAt`, `updatedAt`, `source`.

Не удаляйте строки физически: задаче назначается `cancelled`, проекту — `archived`. Технический столбец `syncHash` скрыт; он позволяет Apps Script обнаружить изменение, сделанное вручную или ChatGPT, и обновить метку времени.

## Конфликты и безопасность

- запись с более новым `updatedAt` имеет приоритет;
- секрет хранится локально в `data/google-sheets-sync.json`, который исключён из Git;
- сетевой обмен с Google выполняется только по HTTPS;
- для смены секрета удалите свойство `ORBITA_SYNC_SECRET` в Apps Script, снова запустите `setupOrbita` и повторите настройку устройств;
- не храните пароли, ключи и иные секреты в описаниях задач.
