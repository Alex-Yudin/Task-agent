## Принятые решения

1. Ядро не зависит от конкретного UI или поставщика ИИ.
2. Локальный анализатор обеспечивает работоспособность без API-ключей и интернета.
3. Все операции изменения данных проходят доменную валидацию и фиксируются в журнале.
4. SQLite настроена на восстановление после аварии и строгую синхронизацию записи.
5. Сущности контекста, документа, контакта, события, решения, напоминания и источника объединены в расширяемое хранилище знаний с типом `kind`.

## Следующая итерация 0.5

1. Карточка задачи с редактированием всех полей и связей.
2. Надёжная обработка сложных составных команд через подключаемый LLM-провайдер.
3. Напоминания Windows и календарное представление.
4. Шифрование чувствительных значений через Windows DPAPI.
5. Автоматическое резервное копирование по расписанию и восстановление из интерфейса.
6. Сборка автономного `.exe` и установщика без требования отдельно ставить Node.js.
7. История конфликтов и корзина удалённых записей в Google Sheets.
